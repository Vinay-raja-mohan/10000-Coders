# 10000-Coders Learning Repository

A comprehensive collection of sample exercises and projects organized by technology and learning day.

## 📁 Repository Structure

### Backend Technologies
- **BackEnd-Technologies/** — Backend frameworks, server-side development, and API design exercises

### Databases
- **DataBases/MYSQL/** — MySQL course materials with daily lessons and assignments
  - Daily folders: `DAY-1` through `Day-21`
  - Includes assignments from dashboard and demo-day materials

### Frontend Technologies
- **FrontEnd-Technologies/** — Web development and UI frameworks
  - **HTML/** — HTML fundamentals with day-based lessons
  - **CSS/** — Styling and layout exercises
  - **JS/** — JavaScript programming exercises
  - **REACT/** — React framework and component-based development

### Scripting Languages
- **Scripting-Language/Python/** — Python programming exercises with daily lessons

### Projects
- **Projects/Mini-Projects/** — Hands-on projects including:
  - Calculator
  - And more...

### Competitive Programming
- **TCS NQT/Coding/** — TCS NQT coding interview preparation

## 🚀 Getting Started

### Running Python Scripts
Execute Python lessons directly from your terminal:

```powershell
python Scripting-Language/Python/Day-3/Day_3.py
```

### File Organization
Each lesson typically includes:
- **Code file** (e.g., `Day_3.py`) — Executable script
- **Notes file** (e.g., `Day_3.txt`) — Solutions and explanations

## 📚 Learning Path

Follow the day-based structure within each technology folder to progress through lessons systematically.
